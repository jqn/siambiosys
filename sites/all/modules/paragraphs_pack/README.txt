# Overview

Paragraphs pack is a collection of common paragraph types. Each type is separated into its own sub-module. To use a paragraph type just enable one of the sub-modules.

# Paragraph types

- "Content"
  Basic paragraph type with just a long text field.

- "Content list"
  This paragraph type allows you to list node content and select which view mode should be used.

- "Taxonomy term list"
  This paragraph type allows you to list term content and select which view mode should be used.

# Requirements

- Paragraphs
